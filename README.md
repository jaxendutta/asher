# Asher Kim - Research Portfolio
Portfolio website for Asher Kim, MSc Biology Candidate at the University of Ottawa.

## Quick Start
```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Tech Stack
- Next.js 16
- React 19
- TypeScript 5
- Tailwind CSS 4

## Updating Content
Edit files in `/src/data/`:
- `education.ts` - Education and awards
- `research.ts` - Research experience
- `publications.ts` - Publications
- `talks.ts` - Talks and presentations
- `outreach.ts` - Outreach activities
- `siteConfig.ts` - Contact info and site metadata

## Deployment
Deployed on Vercel. Push to main branch to deploy.
